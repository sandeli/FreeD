📺 Freed - 一个记录TVBOX源的Github项目
Freed 是一个记录TVBOX源的Github项目，旨在为TVBOX用户提供更新的源信息。

💻 功能特性
收集TVBOX可用源，如：

提供稳定的源
提供稳定的更新
提供新的源
根据品牌、型号分类，方便查找

定期更新，为TVBOX提供最新的源信息

🎨 技术栈
Freed 使用了以下技术实现功能：

HTML
CSS
JavaScript
GitHub API
📝 如何使用
在项目主页中 Freed 选择你想要的TVBOX源
根据品牌、型号查找源信息
使用提供的源信息更新TVBOX
